# Pokedex

## Project Description
Pokedex is a comprehensive Pokémon database application that provides extensive information about various Pokémon, their abilities, and characteristics. This application allows users to search for Pokémon, view their details, and explore various aspects of the Pokémon universe.

## Features
- Search for Pokémon by name
- View detailed information about each Pokémon
- Filter Pokémon by type
- Responsive design for various devices

## Technologies
- **Frontend:** React, Redux, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/hartur-sales/pokedex.git
   cd pokedex
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Usage
- Open your web browser and go to `http://localhost:3000` to view the application.
- Use the search bar to find your favorite Pokémon and explore their details.
